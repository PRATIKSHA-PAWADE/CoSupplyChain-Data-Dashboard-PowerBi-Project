CoSupplyChain Data Dashboard

An interactive Power BI dashboard designed to analyze and visualize supply chain performance.
It provides insights into shipping, sales, customers, products, and market trends to support data-driven decisions.

 Features

Multi-page dashboard with page navigation

Dynamic KPIs and visual analytics

Insights on shipping performance, customer segmentation, and product categories

Tools Used

Power BI

Microsoft Excel / CSV Dataset

 Key Insights

Identifies top-performing regions and categories

Tracks late deliveries and shipping efficiency

Highlights customer and sales trends
